# Vue DateRangePicker

A simple and customizable date range picker component for Vue 3.

## Installation

```bash
npm install vue-daterangepicker
```

## Usage

```vue
<template>
  <DateRangePicker />
</template>

<script>
import DateRangePicker from "vue-daterangepicker";
import "vue-daterangepicker/dist/style.css";

export default {
  components: {
    DateRangePicker,
  },
};
</script>
```

## Development

```bash
# Install dependencies
npm install

# Run demo
npm run dev

# Build library
npm run build
```
